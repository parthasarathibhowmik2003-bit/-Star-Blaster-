# -Star-Blaster-
I've created NEON DEFENDER — a fully-featured space shooter with:
🎮 Core Features:
Smooth Controls: Arrow keys or WASD to move, Space to shoot, P to pause
Progressive Difficulty: Waves get harder as your score increases
Enemy Variety: Basic (pink), Tank (purple hexagons with HP), and Fast (yellow diamonds)
Visual Effects: Neon glows, particle explosions, engine trails, animated stars
Health System: 3 lives with heart display
Score Tracking: Persistent high score saved locally
🎨 Visual Design:
Cyberpunk neon aesthetic (cyan & magenta)
Glowing UI elements with animations
Dynamic wave announcements
Responsive canvas that scales with window
⚡ Technical Polish:
60fps game loop
Collision detection
Screen shake feel through particle effects
Pause menu with resume/quit options
Game over screen with final stats
